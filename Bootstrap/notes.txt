Bootstrap
*********

Bootstrap is a external CSS library or Frameworks

Bootstrap is open source Frameworks

How to use Bootstrap ?
 ->we directly give link in header section to use it.
 ->we directly give link in end of the body section use Bootstrap script.
Include via CDN(content delivery network)

we need to include Bootstrap's compiled CSS

https://getbootstrap.com/

Add header section:-
<link href="https://cdn.jsdelivr.net/npm/bootstrap@5.3.3/dist/css/bootstrap.min.css" 
rel="stylesheet" integrity="sha384-QWTKZyjpPEjISv5WaRU9OFeRpok6YctnYmDr5pNlyT2bRjXh0JMhjY6hW+ALEwIH" crossorigin="anonymous">

We can add component HTML based on doc:-
https://getbootstrap.com/docs/5.3/components/card/

our custom CSS link come to below bootstrap link.
